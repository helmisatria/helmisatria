Kamu bisa pakai google colab diatas kalau butuh script python di google colab nya untuk contoh penggunaan tweet-harvest yang paling baru (versi 2.6.0)

Perbedaan dari versi yang sebelumnya:

- CSV menggunakan separator koma (,) sehingga visualisasi datanya lebih mudah
- Sudah otomatis menunggu jika terkena rate limit tiap sekitar 10-15 menit
- Sudah bisa melakukan SNA (Social Network Analysis)
- Sudah di test untuk mendapatkan data lebih dari 2500 tweets

Code atau script crawl data twitter (tweet-harvest) bisa kalian cek di github: [helmisatria/tweet-harvest: Scrape tweets from Twitter search results based on keywords and date range using Playwright. Save scraped tweets in a CSV file for easy analysis](https://github.com/helmisatria/tweet-harvest)
