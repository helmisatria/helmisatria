Udah dicoba juga untuk mendapatkan lebih dari 200.000 data tweet dari twitter, dan berhasil. Berikut adalah script yang saya gunakan untuk mendapatkan data tweet dari twitter.

```
twint --username jokowi --output output.csv --csv --limit 300
```
